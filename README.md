# 👨‍💻 협업방식
1. 모든 에픽, 스토리, 테스크, 작업, 브랜치는 jira를 통해 관리한다.
2. 정기 스크럼을 통해 개발현황을 보고한다.
3. 현재 개발 피처에서 합의가 필요하면 바로 대화한다.
4. git flow를 사용하여 브랜치를 관리한다.
